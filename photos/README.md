# 马十一照片索引

> 这里记录马十一的重要照片与视频索引。

照片不只是影像，也是成长档案的一部分。每一张被选入索引的照片，都应该有一个清楚的日期、描述，以及它值得保存的原因。

---

## 照片命名规范

推荐格式：

```text
YYYY-MM-DD_day-XXX_简短描述.jpg
```

示例：

```text
2026-06-26_day-000_睡在斗篷里.jpg
2026-06-27_day-001_第一次正式称重.jpg
```

---

## 照片索引模板

```markdown
### YYYY-MM-DD｜Day X｜照片标题

- **文件名**：
- **地点**：
- **描述**：
- **为什么值得保存**：
```
