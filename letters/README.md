# 给马十一的信

> 这里保存妈妈写给小十一的信。

日记记录每天发生了什么。

信件记录妈妈想对小十一说什么。

它们不需要频繁，也不需要很长。可以是一年一封，也可以是在某个特别的日子里，写下几句话。

---

## 写信原则

- 不追求完整，只追求真实。
- 不需要每天写。
- 可以写给现在的小十一，也可以写给未来的小十一。
- 可以很短。
- 可以很普通。
- 只要是妈妈真正想说的话，就值得保存。

---

## 推荐命名

```text
YYYY-MM-DD_给小十一的一封信.md
```

示例：

```text
2026-06-26_欢迎回家.md
2027-06-26_一周年.md
```

---

## 信件模板

```markdown
# 给小十一的一封信

日期：YYYY-MM-DD  
Day：Day X

小十一：

（妈妈想说的话）

—— 妈妈
```
